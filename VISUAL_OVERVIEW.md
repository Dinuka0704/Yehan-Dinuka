## 🔄 Component Flow

```
App.js
  │
  ├─→ Header (Fixed Navigation)
  │     └─→ Logo + Navigation Menu + Hamburger
  │
  ├─→ Hero (Home Section)
  │     ├─→ Text Content
  │     │    ├─→ Greeting
  │     │    ├─→ Name
  │     │    ├─→ Typewriter Effect
  │     │    ├─→ Social Links
  │     │    └─→ Statistics
  │     └─→ Hero Image
  │
  ├─→ About (About Section)
  │     ├─→ Text + Image
  │     ├─→ Programming Languages Grid
  │     ├─→ Frameworks Grid
  │     ├─→ Tools Grid
  │     ├─→ Databases Grid
  │     └─→ Experience Stats
  │
  ├─→ Education (Education Section)
  │     ├─→ Timeline Items
  │     │    ├─→ Degree
  │     │    ├─→ Institution
  │     │    ├─→ Period
  │     │    └─→ Achievements
  │     └─→ Certifications Grid
  │
  ├─→ Projects (Projects Section)
  │     ├─→ Projects Grid
  │     │    ├─→ Project Card
  │     │    │    ├─→ Image + Overlay
  │     │    │    ├─→ Title + Description
  │     │    │    ├─→ Technologies
  │     │    │    └─→ Links (GitHub, Demo)
  │     └─→ CTA Section
  │
  ├─→ Contact (Contact Section)
  │     ├─→ Contact Info
  │     │    ├─→ Image
  │     │    └─→ Contact Details
  │     └─→ Contact Form
  │          ├─→ Name Input
  │          ├─→ Email Input
  │          ├─→ Subject Input
  │          ├─→ Message Textarea
  │          └─→ Submit Button
  │
  └─→ Footer (Footer Section)
        ├─→ Logo + Description
        ├─→ Quick Links
        ├─→ Contact Info
        ├─→ Social Media Links
        ├─→ Copyright
        └─→ Scroll to Top Button
```
